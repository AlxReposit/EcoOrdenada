# EcoOrdenada

Esse site é um projeto de extensão para o curso de Análise e Desenvolvimento de Sistemas da Universidade Cruzeiro do Sul. O objetivo é incentivar a economia circular e o desenvolvimento sustentável pela conscientização sobre reciclagem e descarte correto de resíduos, e melhorar o acesso à informação de locais que oferecem esses serviços.

### OpenStreetMap

Foram utilizados dados da OpenStreetMap (OSM - https://www.openstreetmap.org/about), que é “um projeto de produção colaborativa de dados geo-espaciais abertos”, disponibilizando informações de mapas e locais para todos, a partir da contribuição da própria comunidade.

Limitações: A OSM não possui informações de todas as organizações e locais de reciclagem e coleta seletiva. Para algumas regiões, pode até não haver informação nenhuma.
              
Para contornar essa situação, o site EcoOrdenada utiliza um banco de dados próprio com locais inseridos manualmente em conjunto com os dados do Open Street Map.

Se desejar ajudar a expandir o mapa da Open Street Map e acrescentar pontos e locais, acesse este guia de introdução: https://www.openstreetmap.org/about


### Instruções

Para encontrar locais de reciclagem e descarte, digite o nome de uma cidade ou endereço na caixa de Busca, a distância da busca em quilômetros e clique no botão com a lupa. Outra forma de realizar a busca é mover o mapa manualmente e clicar no botão “Buscar no Mapa” no canto inferior esquerdo para procurar todos os pontos que vão aparecer na área que o mapa está mostrando.
               
Os locais encontrados vão aparecer como pontos no mapa, e como itens na tabela lateral, que podem ser clicados para serem visualizados no mapa. Também é possível clicar no link para abrir a coordenada no google maps, mas nem sempre o local procurado vai estar indicado no google Maps, ou pode estar em uma coordenada próxima.
